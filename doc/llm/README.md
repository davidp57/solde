# Solde ⚖️ — LLM Reference

## Contents

- [reference.md](./reference.md) — User-assistance knowledge base for an LLM chatbot

This folder contains the knowledge base for an LLM chatbot that assists end users of the Solde application.
It covers all user workflows, UI navigation, roles and permissions, terminology, business rules, and a comprehensive Q&A section for common user questions.

This is **not** a technical or developer reference. It is written from the user's perspective, using the application's French terminology.

**Language:** English.
